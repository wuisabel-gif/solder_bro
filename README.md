# 🔥 Solder Bro

Solder Bro is a single-page visual troubleshooting guide for soldering. Instead of digging through forums or datasheets, just match what your solder joint looks like to the defect you're seeing, learn why it happened, and find the fastest way to fix it.

Built for students, makers, hobbyists, and anyone who has ever stared at a PCB wondering, *"Why won't this pad wet?"*

## 🛠 Features

### 👀 Read the Joint
Start with a textbook solder joint. An annotated reference fillet shows what a healthy joint should look like, making it easier to spot what's wrong when things go sideways.

### 🔍 Symptom-Based Diagnosis
Filter defects by what you actually see:

- Dull or grainy joints
- Cracks
- Solder bridges
- Poor wetting
- Burnt pads
- Excess solder
- Missing solder
- Components standing upright ("tombstoning")
- And more

No electronics background required — start from the symptom and work backward.

### ⚡ Common Soldering Defects
Includes visual examples of:

- Cold joints
- Oxidized joints
- Solder bridges
- Non-wetting
- Overheated joints
- Tombstoning
- Lifted pads
- Insufficient solder
- Excess solder
- Disturbed joints
- Flux contamination

Each defect includes:

- What it looks like
- Why it happens
- How to fix it
- How to prevent it next time

### 🧰 Tool Finder
Not sure what tool you need? Browse equipment by task and see exactly which defects each tool helps solve. From flux pens and wick to hot-air stations and tip tinners, every recommendation is tied to a real repair scenario.

### 🎥 Learn by Watching
Embedded beginner-friendly soldering tutorials from trusted sources such as:

- Adafruit
- EEVblog

Watch repairs and techniques directly inside the app.

## 🚀 Getting Started

No installation required. Simply open:

```text
solder-joint-clinic.html
```

in any modern web browser.

You can also deploy it as a static site using:

- GitHub Pages
- Netlify
- Vercel
- Any static web host

An internet connection is required for web fonts, embedded videos, and external media.

## ⚙️ Built With

- HTML5
- CSS3
- Vanilla JavaScript
- Hand-drawn SVG illustrations

No frameworks. No build tools. No dependencies. Every solder joint illustration is rendered as SVG rather than image assets, keeping the project lightweight and scalable.

## 📚 Disclaimer

Solder Bro is intended as an educational and troubleshooting reference. For professional manufacturing and inspection standards, refer to the appropriate IPC specifications, including IPC-A-610.

## 🤝 Contributing

## 👨‍💻 Why I Built This

I'm an engineering student, and soldering problems show up constantly:

- Cold joints that look fine until they fail
- Random solder bridges
- Pads that refuse to wet
- Components that mysteriously tombstone
- Rework sessions that take longer than the original assembly

After repeatedly searching through forums and YouTube videos for the same answers, I decided to build a visual reference that puts the diagnosis, explanation, fix, and tools all in one place.

If Solder Bro helps you save a board — or at least avoid a few unnecessary reflows — then it's doing its job.
